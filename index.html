Simon Memory Game

Instructions:

  • Simon will give the first signal. Repeat the signal by pressing the same colour lens.
  • Simon will continue to add signals to the sequence. Each turn, you must repeat the entire sequence in order and then add on the latest signal.

    • If you fail to repeat the sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
